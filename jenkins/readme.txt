Создание задания Pipeline в Jenkins:
  1. На главной странице Jenkins нажмите «New Item» («Создать задание»).
  2. Введите название для задания.
  3. В списке типов заданий выберите «Pipeline».
  4. Нажмите «OK».
  5. Из выпадающего списка настройки задания поля "Definition" выберите опцию "Pipeline script from SCM" (говорит Jenkins брать конфигурацию файла Jenkinsfile).
  6. В поле "SCM" выберите систему контроля версий, например, Git.
  7. В поле "Repository URL" вставьте ссылку на репозиторий с кодом (https://github.com/levvolkov/appium-selenide-mobile-tests).
  8. В поле "Script Path" укажите путь к Jenkinsfile (если он в корне проекта, тогда по умолчанию - Jenkinsfile).
  9. Нажмите "Save" для сохранения настроек.
  10. Вернитесь в список заданий и нажмите "Build Now" для первого запуска.
